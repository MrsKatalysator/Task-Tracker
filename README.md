Task-Tracker

A customizable task tracker for private use. Developed as Web-App :) My goal was to get used to object-oriented programming as well as to practice solving problems that come with it. My main focus is usually Python programming, but I wanted to try to develope a frontend project to get used to publishing and to get to know the base constructs for frontend before I dive deeper into backend with Python. It's also one of my first projects I uploaded to Github. Hopefully someone besides me can use it :) 

## Feature Overview

### Task Management:
• Create tasks with name, project, priority, due date and tags 
• Check off tasks (mark as done) 
• Delete tasks 
• Set priorities: Low 🟢, Medium 🟡, High 🔴 
                
### Projects:
• Project dropdown showing all projects 
• Create new projects directly in the dropdown (press Enter) 
• Delete projects (except "General")
• Drag & Drop tasks between projects 
        
### Subtasks:
• Add subtasks to any task 
• Progress bar shows completion automatically 
• When all subtasks are done → main task is auto-completed 
        
### Tags:
• Tags can be added to tasks (comma-separated) 
• Tag filter: click a tag to see only tasks with that tag 
        
### Drag & Drop:
• Move tasks between projects via drag & drop 
• Reorder by dragging to another task 
        
### Search & Filters:
• Full-text search across tasks, subtasks and tags 
• Filters for All / Open / Done / Overdue 
        
### Language:
• German/English toggle (via 🇩🇪/🇬🇧 button) 
        
### Settings:
• 5 accent colors: Purple, Blue, Green, Orange, Pink 
• 3 font sizes: Small, Medium, Large 
• Dark Mode & light Mode 
        
### Technical:
• All data stored locally in the browser (localStorage) 
• No login required 
• Mobile app-ready (Web App Manifest + icons) 
• Responsive design
